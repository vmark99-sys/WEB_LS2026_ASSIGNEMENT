# GSD Website

This project is a static web portal about German Shepherd dogs.  
The website contains a home page, several subpages, a gallery with image preview, and a contact form with client-side validation.

## Live website

The website is available here:

https://vmark99-sys.github.io/WEB_LS2026_ASSIGNEMENT/

## Project content

The website includes:

- Home page
- About page
- Contact page with form validation
- Galery page with JavaScript image preview
- News / article subpages
- Custom CSS styling
- Images and logo used across the website

## Technologies used

- HTML5
- CSS
- JavaScript
- GitHub Pages

## How to run the project

1. Download or clone the repository:

git clone https://github.com/vmark99-sys/WEB_LS2026_ASSIGNEMENT.git

2. Open the project folder.

3. Open the file index.html in a web browser.

The website does not require any installation, database, or server because it is a static website.
